### Development

#### Frontend:
Install Node.js, Vue CLI.<br>
Run:
```
/src/frontend: npm install
/src/frontend: npm run serve
```
Then connect to `localhost:8080`.
#### Backend:
Install MongoDB, JDK 11.<br>
Run:
```
/: mvn package
```
Then navigate to /target and run the jar there using:
```
java -jar <filename>
```
